## Para quien es Owli-AI Zoom?

Owli-AI Zoom esta dirigida a personas con discapacidad visual que desean adaptar mejor su puesto de trabajo en escritorio a sus necesidades visuales.

Zoom combina ampliacion de pantalla en Windows con una lupa de escritorio practica para trabajo y estudio. Ademas de lectura en voz TTS bajo el puntero, incluye un magnificador de pantalla completa para sesiones largas y concentradas.

Ya sea para leer textos, trabajar con tablas o navegar por aplicaciones, Zoom te ayuda a percibir el contenido con mayor claridad.

## Que hace la app exactamente?

Owli-AI Zoom amplia en tiempo real zonas seleccionadas de tu pantalla. Ademas, ofrece varios modos de contraste y color para mejorar la visibilidad.

La ampliacion puede ajustarse de forma flexible para configurar el nivel de zoom que mejor te funcione.

## Funciones de lectura integradas

Owli-AI Zoom no solo ayuda con ampliacion, tambien incorpora funciones de lectura.

El texto bajo el puntero puede leerse automaticamente. Como alternativa, puedes hacer clic en fragmentos concretos y reproducirlos por voz.

Asi puedes procesar contenido mas rapido sin descifrar todo visualmente. En documentos largos, sitios web o tablas, esto puede aumentar la velocidad de lectura y reducir la fatiga visual.

## Modo de pantalla completa

Ademas de la vista lupa clasica, hay un modo de ampliacion de pantalla completa. En este modo se muestra toda la pantalla ampliada.

Este modo es especialmente adecuado para sesiones largas de trabajo en escritorio, ya que no hay que seguir una zona de lupa separada.

## Trabajo de escritorio con mejor vision global

Zoom se ha desarrollado para uso continuo en escritorio. La ampliacion se mantiene estable mientras cambias entre programas o usas distintas aplicaciones.

Asi puedes usar tu ordenador de forma mas comoda en el dia a dia o en el trabajo.

## Soporte de plataforma

Owli-AI Zoom esta disponible actualmente para Windows.

Hay una version para Linux en planificacion para ofrecer soporte comparable tambien alli.

## En desarrollo

Owli-AI Zoom esta en desarrollo activo. Entre las funciones planificadas se incluyen:

- Analisis asistido por IA del contenido en pantalla  
- Funciones de asistencia ampliadas similares al sistema cloud  
- Un sistema integrado de lectura con camara para documentos y texto  
- Soporte para Linux  

Estas funciones se encuentran actualmente en fase de concepto y desarrollo.

## Privacidad y procesamiento

Owli-AI Zoom procesa los datos de imagen exclusivamente de forma local en tu dispositivo. No se transfieren datos de pantalla a servidores externos.
